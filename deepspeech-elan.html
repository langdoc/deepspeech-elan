DeepSpeech Phoneme Recognizer

This is a local recognizer that offers automatic speech recognition using Mozilla's DeepSpeech, an open-source automatic speech recognition tool. The user is able to select a tier in the current transcript, and provides the path to DeepSpeech model (pb or pbmm formats), and to an external language model scorer. This recognizer then processes the contents of all segments on the specified tier, producing a new tier containing the automatically recognized strings that DeepSpeech has generated.
